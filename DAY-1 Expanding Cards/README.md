# Expanding Cards

A responsive image gallery with smooth expanding animations built with vanilla HTML, CSS, and JavaScript.

## Features

- Click any card to expand it with smooth transitions
- Responsive design that works on mobile devices
- Beautiful background images with overlay text
- Clean, modern UI with rounded corners

## Technologies Used

- HTML5
- CSS3 (Flexbox, Transitions)
- Vanilla JavaScript

## How to Use

1. Clone or download the project
2. Open `index.html` in your browser
3. Click on any card to see the expanding animation

## Live Demo

Simply open the `index.html` file in any modern web browser to see the project in action.

## Responsive Design

The project includes mobile-responsive features:
- On screens smaller than 480px, only 3 cards are displayed
- Layout adjusts to fit different screen sizes
