# Progress Steps

An interactive progress indicator with step navigation built with vanilla HTML, CSS, and JavaScript.

## Features

- Click "Next" to advance through steps
- Click "Prev" to go back to previous steps
- Animated progress bar that fills as you advance
- Button states automatically disable when at first/last step
- Smooth CSS transitions for visual feedback

## Technologies Used

- HTML5
- CSS3 (CSS Variables, Flexbox, Transitions)
- Vanilla JavaScript

## How to Use

1. Open `index.html` in your browser
2. Click "Next" to move forward through the steps
3. Click "Prev" to move backward through the steps
4. Watch the progress bar animate and circles activate
